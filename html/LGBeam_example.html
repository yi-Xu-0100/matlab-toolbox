LGBeam.m

LGbeam(p, l, w_0, z, lambda, rho, theta) 是一个复数标量函数，主要用于计算拉盖尔-高斯光束（LGBeam）场振幅的分布[1,2]。

基本参数

LGBeam 函数描述了满足准轴近似下的波动方程并受以下输入参数限制的拉盖尔-高斯光束场振幅分布：
p = 0; % 径向量子数
l = 0; % 角向量子数
w_0 = 5e-4; % 光束束腰尺寸
z = 0; % 传播距离
lambda = 5e-7; % 光束波长
而为了获得柱坐标系下的半径和角度，可以使用 cart2pol(x, y) 函数
L = 10e-3;
N = 1000;
xoy = linspace(-0.5 * L, 0.5 * L, N);
[X, Y] = meshgrid(xoy, xoy);
% 获取柱坐标系下，半径 rho 和角度 theta
[theta, rho] = cart2pol(X, Y);
